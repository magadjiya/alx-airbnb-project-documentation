# User Stories Based on [Use Case Diagram](../use-case-diagram)

## Users
*User Registration*: 
As a user, I want to register an account so that I can access the platform and use its features.

*Login*: As a user, I want to log into my account so that I can securely access my dashboard.

*Manage Profile*: As a user, I want to manage my profile so that my personal information stays updated.

*Send/Receive Messages*: As a user, I want to send and receive messages so that I can communicate with hosts or guests.

*Receive Notifications*: As a user, I want to receive notifications so that I can stay updated about bookings and messages.

*Logout*: As a user, I want to log out so that my account remains secure.

## Guest
*Book Property*: As a guest, I want to book a property so that I can reserve it for my stay.

*Make Payment*: As a guest, I want to make secure payments so that my booking can be confirmed.

*Write Review*: As a guest, I want to write reviews so that I can share feedback about my stay.

## Host
*Create Listing*: As a host, I want to create property listings so that guests can book my property.

*Edit Listing*: As a host, I want to edit my property listing so that I can keep the details accurate.

*Delete Listing*: As a host, I want to delete my property listing so that it is no longer visible to guests.

*Manage Bookings*: As a host, I want to manage bookings so that I can prepare for incoming guests.

*Reply Reviews*: As a host, I want to reply to guest reviews so that I can address feedback and maintain a good reputation.

## Admin
*Manage Users*: As an admin, I want to manage user accounts so that I can maintain platform safety.

*Manage Listings*: As an admin, I want to manage listings so that I can remove inappropriate or fraudulent properties.

*View Bookings*: As an admin, I want to view all bookings so that I can monitor platform activity.

*Moderate Reviews*: As an admin, I want to moderate reviews so that inappropriate reviews can be removed.
